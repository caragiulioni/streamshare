# Streamshare

![Streanshare](resources/landing.png)

Search movies, TV and video games.

Add titles to your list.

Follow friends and find new titles.

## Installation

```bash
cd server

yarn install

yarn dev
```

```bash
cd client

yarn install

yarn start
```
